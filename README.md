# Гармония — помощник ЦСОН (ЯНАО)

Веб-приложение в виде чат-бота для ГБУ ЯНАО «ЦСОН Гармония»: прейскурант
социальных услуг, корзина и заявки, запись к специалисту, обратный звонок,
соцработник на дом, мероприятия, калькулятор льгот, умный помощник, личный
кабинет и админпанель. Поддерживает 5 филиалов с индивидуальными данными.

## Полная документация

Подробное руководство (структура, публикация на GitHub Pages, редактирование
данных, админпанель, подключение базы, ФЗ-152, памятка администратора) — в файле:

**[ДОКУМЕНТАЦИЯ.md](./ДОКУМЕНТАЦИЯ.md)**

## Кратко

- Открыть локально: двойной клик по `index.html` (или `python3 -m http.server`).
- Публикация: GitHub → загрузить файлы → Settings → Pages → ветка `main`, `/ (root)`.
- Данные (услуги, цены, сотрудники, контакты, новости) — в `js/data.js`.
- Админпанель: ссылка на экране входа, логин/пароль — у ответственного за сайт.
- Все заявки уходят на `cson-gub@yanao.ru`.

## Структура

```
index.html · css/styles.css
js/ data.js · db.js · features.js · admin.js · app.js
data/ services.json · staff.json · news.json · events.json · faq.json · branches/
```

## Важно

Сайт работает без сервера: заявки уходят на почту, правки админа хранятся в его
браузере. Для общей базы, защищённого входа и онлайн-записи нужно подключение
Supabase — см. раздел 9 документации. Сбор СНИЛС регулируется ФЗ-152 (раздел 11).
